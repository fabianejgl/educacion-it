PSEUDOCLASES

Son selectores en css que permiten aplicar estilos a elementos específicos en ciertos estados o condiciones. Las pseudoclases se añaden a los selectores css (.clase, #id, etiqueta) se le agrega la pseudoclase (clase:pseudoclase, #id:pseudoclase, etiqueta:pseudoclase)
