Formularios en HTML

Para crear un fomrulario usamos la etiqueta "form> /form>"

Atributos de la etiqueta form

  • Action: Es la ruta (url) a donde se va a enviar la información recopilada
  • Method: (protocolo http => metodos http) - Método GET y POST. SOLO A TRAVES DE ESTAS 2 FORMAS PUEDO ENVIAR LA INFORMACION. Es como se va a enviar la información. GET => Através de la URL o POST => A través del cuerpo de HTML
  • Target: El mismo comportamiento que tenía en los links (anchor tags). Si le coloco target = "_blank" se abre en otra pestaña. Si le coloco target="_self" se abre en la misma pestaña.

Formulario al enviarse va a armar la URL (Query String)

Cuando envío la información a través del método GET. El formulario automáticamente arma la URL para enviar la información. A esa URL se llama (Query String)

Ejemplo: www.servidor(backend).net/?nombredevalor1=valor1&nombredevalor2=valor2

Atributos INPUT

  • type
  • name
  • placeholder
