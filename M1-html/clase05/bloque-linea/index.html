Elementos de línea vs elementos de bloque

Elementos en línea (inline)

Se utiliza pra representar contenido en línea, es decir, contenido que se muestra dentro del flujo nomal del texto. Estos elementos inician en una nueva línea y solo ocupan el espacio necesario de su contenido

Elementos de bloque

Se utilizan para crear secciones o bloques de contenido distintos en la página. Estos elementos siempre inician en una nueva línea y ocupan todo el ancho disponible de su contenedor principal
